Menu : clipboard <-> file

The clipboard <-> file button can be used to transfer text data between the internal or external clipboard and a local or remote file. You must first be connected to a remote host for this button to work.

Once connected, click this button. A menu will appear with these 4 options:

  • Internal clipboard -> File
  • External clipboard -> File
  • File -> Internal clipboard
  • File -> External clipboard

The first two options write one of the clipboards to a new file. After clicking one of those buttons, navigate to the directory you want to create the new file in, then click the clip -> file button at the top. You will be prompted to enter the name of the file you want to create. Be sure to enter a name that doesn't already exist in the directory, or you will receive a warning. Click the OK button and the file will be created with the clipboard written to it.

The last two options read an existing file into one of the clipboards. After clicking one of those buttons, navigate to the directory the file is in and double-click on the file. Then click the file -> clip button at the top. There must be exactly one file selected and that file must be readable for the file -> clip button to appear. The file is now in the clipboard.
